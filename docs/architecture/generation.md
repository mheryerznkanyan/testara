# Test Generation

How Testara generates XCUITest code from plain English.

## Process

1. **Enrichment:** Expand vague descriptions
2. **Context Retrieval:** Find relevant code
3. **Prompt Construction:** Build LLM prompt
4. **Generation:** Claude creates test code
5. **Validation:** Syntax and quality check
