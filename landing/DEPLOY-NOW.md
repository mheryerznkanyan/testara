## ⚡ Quick Deploy (5 Minutes)

### Step 1: Install Dependencies
```bash
cd landing
npm install
```

### Step 2: Run Development Server
```bash
npm run dev
```

### Step 3: Open in Browser
```
http://localhost:3000
```

**That's it!** The landing page is now running.
